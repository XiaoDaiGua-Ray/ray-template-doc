# 安装步骤

## 获取项目

```
git clone https://github.com/XiaoDaiGua-Ray/xiaodaigua-ray.github.io.git
```

## 依赖安装

```sh
# yarn

yarn
```

```sh
# npm

npm install
```

::: danger 提示
尽量使用 yarn 安装依赖，避免使用 cnpm 安装，会出现诡异的问题。
:::

## 启动项目

```sh
# yarn

yarn dev
```

```sh
# npm

npm run dev
```

::: warning 提示
由于本项目基于 vite 作为构建工具，首次构建时间会稍微长一点。
:::
