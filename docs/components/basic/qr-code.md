# Vue3NextQrcode 二维码

显示一个有点意思的二维码。

## 说明

该组件是由 `RQRCode` 组件抽离出去单独发布的一个二维码组件库。

抽离出去，是为了解决本地编译环境安装繁琐的问题，同时也方便其他项目使用。
