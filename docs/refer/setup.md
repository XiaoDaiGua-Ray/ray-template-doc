# 快速上手

准备，启航！

## 命令

一些常用的相关命名。

::: code-group

```sh [安装依赖]
pnpm i
```

```sh [启动项目]
pnpm dev
```

```sh [构建项目]
pnpm build
```

```sh [代码检查与修复]
pnpm lint
```

```sh [体积分析]
pnpm report
```

```sh [单测]
# 命令行查看
pnpm test

# 可视化查看
pnpm test:ui
```

:::
